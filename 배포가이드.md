# 🚀 숙명여중 2-5반 학부모 공지 앱 — 배포 가이드

## 전체 흐름 요약
Supabase (DB) → GitHub (코드) → Vercel (배포) → 카카오톡 공유

---

## STEP 1. Supabase 프로젝트 생성 및 DB 구성

### 1-1. 프로젝트 생성
1. https://supabase.com 접속 → 로그인
2. **New project** 클릭
3. 이름: `school-25ban`, 비밀번호 설정, 지역: **Northeast Asia (Seoul)** 선택
4. 약 2분 대기 후 프로젝트 생성 완료

### 1-2. DB 스키마 + 초기 데이터 입력
1. 왼쪽 메뉴 **SQL Editor** 클릭
2. `supabase/schema.sql` 파일 전체 내용을 복사
3. SQL Editor에 붙여넣기 → **Run** 클릭
4. ✅ 완료 메시지 확인

### 1-3. API Key 복사 (나중에 사용)
- 왼쪽 메뉴 **Settings → API**
- `Project URL` 복사 → 메모장 저장
- `anon public` 키 복사 → 메모장 저장

---

## STEP 2. 로컬 개발 환경 실행 (선택)

```bash
# Node.js 18+ 필요 (https://nodejs.org)

# 1. 프로젝트 폴더로 이동
cd school-app

# 2. .env 파일 생성
cp .env.example .env
# .env 파일 열어서 Supabase URL, ANON_KEY, 관리자 비밀번호 입력

# 3. 패키지 설치
npm install

# 4. 개발 서버 실행
npm run dev
# http://localhost:5173 에서 확인
```

---

## STEP 3. GitHub에 코드 업로드

1. https://github.com 로그인 → **New repository**
2. 이름: `school-25ban` (private 또는 public 선택)
3. **Create repository**
4. 터미널에서:

```bash
cd school-app
git init
git add .
git commit -m "초기 배포"
git remote add origin https://github.com/[내아이디]/school-25ban.git
git push -u origin main
```

---

## STEP 4. Vercel 배포

### 4-1. 프로젝트 연결
1. https://vercel.com 로그인 (GitHub 계정 연동)
2. **Add New → Project**
3. `school-25ban` 저장소 선택 → **Import**
4. Framework Preset: **Vite** 자동 감지됨

### 4-2. 환경변수 설정 (핵심!)
**Settings → Environment Variables**에 아래 3가지 추가:

| 변수명 | 값 |
|--------|-----|
| `VITE_SUPABASE_URL` | https://xxxxx.supabase.co |
| `VITE_SUPABASE_ANON_KEY` | eyJhbG... (anon public 키) |
| `VITE_ADMIN_PASSWORD` | 본인이 설정한 비밀번호 |

### 4-3. 배포
1. **Deploy** 클릭 → 약 1분 소요
2. 배포 완료: `https://school-25ban.vercel.app`

---

## STEP 5. 앱 사용 방법

### 👩 학부모 (일반 사용자)
- 링크 접속: `https://school-25ban.vercel.app`
- 공지 확인, 이모지 반응, 댓글 작성 가능
- 🔕 버튼 → 알림 허용하면 새 공지 시 알림 수신

### ⚙️ 관리자 (대표학부모)
- 관리자 페이지: `https://school-25ban.vercel.app/admin/login`
- 설정한 비밀번호로 로그인
- 공지 작성/수정/삭제, 일정 관리, 연락처 관리 가능

---

## STEP 6. 홈화면에 앱 아이콘으로 추가

### 📱 iPhone (Safari에서만 가능)
1. Safari로 앱 URL 접속
2. 하단 공유 버튼(📤) 탭
3. **홈 화면에 추가** 선택
4. 이름 확인 후 **추가** → 홈화면에 아이콘 생성

### 📱 Android (Chrome)
1. Chrome으로 앱 URL 접속
2. 주소창 우측 메뉴(⋮) → **앱 설치** 또는 **홈 화면에 추가**

---

## 이후 운영

- **공지 추가**: `/admin` 로그인 → 공지사항 탭 → + 추가
- **코드 수정 반영**: git push 하면 Vercel이 자동으로 재배포
- **DB 직접 보기**: Supabase → Table Editor

---

## 문제 해결

| 증상 | 해결책 |
|------|--------|
| 화면이 흰색으로 비어있음 | Vercel 환경변수 확인 |
| 공지가 안 불러와짐 | Supabase RLS 정책 확인 (schema.sql 재실행) |
| 관리자 로그인 안됨 | `VITE_ADMIN_PASSWORD` 환경변수 확인 |
| 빌드 실패 | Vercel 로그에서 오류 메시지 확인 |
